# E-Connect IoT Smart Home System

## Overview
This project is a comprehensive IoT Smart Home solution featuring:
*   **Central Server**: Python/FastAPI backend with SQLite/SQLAlchemy.
*   **Web Dashboard**: Frontend for management and control.
*   **Mobile App**: For remote access.
*   **Firmware**: ESP32-based device logic.
